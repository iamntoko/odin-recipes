# Odin Recipes

This is a simple recipe website built as part of The Odin Project curriculum for the Foundations Course. The project focuses on mastering the basic HTML, directory structuring, and Git workflow.

## Live Demo

GitHub Pages: [https://iamntoko.github.io/odin-recipes/](https://iamntoko.github.io/odin-recipes/)

## Technologies Used

- HTML5
- Git & GitHub

## Features

- **HTML5 Boilerplate:** Standard complaint HTML5 structure.
- **Navigation:** Internal linking between the homepage and recipe pages.
- **Multimedia:** Inclusion of images for various dishes.
- **Lists:** Use of ordered and unordered lists for ingredients and steps.

## Lessons Learned

Through this project, I reinforced my understanding of:

- Creating and navigating relative file paths (example, `../images/enchiladas.jpg`).
- Using Conventional Commits to maintain a simple straightforward and clean project history.
- Structuring data using the right semantic HTML elements like `<ul>` and `<ol>`.
